# cnodejs-vue

> cnodejs vue版，使用vuejs + vue-cli + vue-router + vuex进行开发

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```
